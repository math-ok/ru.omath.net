Доказательство

Обозначим второе определение модуля числа x как ∣x∣′|x|'∣x∣′. Покажем, что какой x не возьми, будет выполняться ∣x∣=∣x∣′|x| = |x|'∣x∣=∣x∣′.

Пусть x>0x >0x>0. По классическому определению ∣x∣=x|x| = x∣x∣=x. По второму: ∣x∣′=x|x|' = x∣x∣′=x. То есть ∣x∣=∣x∣′|x| = |x|'∣x∣=∣x∣′.

Пусть x=0x=0x=0. По классическому определению |0| = 0. А вот во втором определении 0 попадает уже под второе условие, то есть |0|' = -0 = 0. Опять имеем |0| = |0|'.

Наконец, пусть x<0x < 0x<0. По классическому определению ∣x∣=−x|x| = -x∣x∣=−x. У второго определения та же ситуация: ∣x∣′=−x|x|' = -x∣x∣′=−x. Получается, что и в этом случае ∣x∣=∣x∣′|x| = |x|'∣x∣=∣x∣′.

Итак, мы рассмотрели все возможные значения для x и во всех случаях ∣x∣=∣x∣′|x| = |x|'∣x∣=∣x∣′. Это и означает, что между двумя определениями нет никакой разницы ■\blacksquare■
